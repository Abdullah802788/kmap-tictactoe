1. From Board to Boolean Variables

To use logic gates, we must convert the 3×3 grid into 1s and 0s. Every cell on the board is represented by exactly two variables: one tracking if it is an X, and one tracking if it is an O.

2. Win Detection via Sum-of-Products

A player wins if they occupy a full row, column, or diagonal. We express this using a boolean formula combining AND (·) and OR (+) logic. There are exactly 8 ways to win.

WIN_X = X0·X1·X2

The AI evaluates this instantly. Every simulation tick, the entire board state physically falls through these logic gates to determine a win status.

4. The Decision Pipeline

The AI does not think ahead. Instead, it runs the board state through 7 strict, cascaded boolean rule modules simultaneously. The highest-priority module that outputs a "1" gets to choose the move.

1. Win (Can I win directly?)
↓
2. Block (Will I lose next turn?)
↓
3. Fork (Create a double threat)
↓
4. Block Fork (Force opponent defense)
↓
5. Center (Take the middle)
↓
6. Corner (Take strategic corners)
↓
7. Side (Take any remaining edge)
